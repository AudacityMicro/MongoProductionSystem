Robot connection

Choose whether the UI talks to a physical robot or the built-in simulator.

Simulated mode allows manual input/output toggles on the Debugging page. Physical mode reads only live RTDE data from the configured robot.

04

CNC telemetry

Read-only PathPilot/LinuxCNC status is collected over SSH. No machine commands are enabled in this stage.

The controller must allow SSH access for this LinuxCNC user. The application runs linuxcnc.stat().poll() remotely and reads its JSON response.

05

Pallet pool

Pool positions are numbered sequentially and each holds one pallet.

Default: 16. Occupied positions must be cleared before reducing this number below them.

05

Robot file access

Use SFTP to browse program files stored on the robot controller.

Robot extensions filter the controller-program button chooser; .urp is the default. Universal Robots defaults are preconfigured: root / easybot. The password is shown and editable here, and is returned to browsers that can access this trusted local-network application.

06

Debug programs

Configure a compact set of controller-program buttons on the Debugging page.

Each button has its own display name, controller filename, and color. Reducing the count hides higher-numbered buttons without deleting their saved configuration.

07

Robot Programs page

Manage controller files through SFTP and optionally open downloaded files in a local editor.

The command runs on this server computer with a downloaded copy of the selected controller file. Default: code for Visual Studio Code.

08

Mill Programs page

Manage PathPilot user program files through the configured CNC SSH connection.

Uses the PathPilot host, SSH port, username, password, and timeout above. All file operations stay inside this program directory. Default: /home/operator/gcode.

09

Fusion 360 tools

Load tool numbers and descriptions from a Fusion 360 tool-library JSON file.

Upload one or more .tools or JSON libraries. The server stores them locally and merges their tool numbers into the Tools page.
