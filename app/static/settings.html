03

Robot connection

Choose whether the UI talks to a physical robot or the built-in simulator.

Simulated mode allows manual input/output toggles on the Debugging page. Physical mode reads only live RTDE data from the configured robot.

04

Pallet pool

Pool positions are numbered sequentially and each holds one pallet.

Default: 16. Occupied positions must be cleared before reducing this number below them.

05

Robot file access

Use SFTP to browse program files stored on the robot controller.

Robot extensions filter the controller-program button chooser; .urp is the default. Universal Robots defaults are preconfigured: root / easybot. The password is shown and editable here, and is returned to browsers that can access this trusted local-network application.

06

Debug programs

Configure a compact set of controller-program buttons on the Debugging page.

Each button has its own display name, controller filename, and color. Reducing the count hides higher-numbered buttons without deleting their saved configuration.
